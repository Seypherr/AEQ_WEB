Déclaration d’accessibilité

Bois Noble s’engage à rendre ce site accessible au plus grand nombre, dans le cadre du projet.

Synthèse

Dernière mise à jour : 06/01/2026

  • Niveau visé : bonnes pratiques WCAG / RGAA (objectif pédagogique).
  • Navigation clavier : menu mobile utilisable au clavier (tabulation + Escape).
  • Structure : en-têtes/landmarks, titres hiérarchisés, lien d’évitement.
  • Formulaires : labels explicites, focus visible, champs requis indiqués.

Fonctionnalités d’accessibilité

  • Lien “Aller au contenu principal”.
  • Focus visible renforcé.
  • Navigation structurée : header/nav/main/footer.
  • Images avec textes alternatifs pertinents.
  • Menu mobile : fermeture par Escape + gestion du focus.

Compatibilité

  • Clavier seul.
  • Lecteurs d’écran (structure et libellés adaptés).
  • Responsive mobile / tablette / desktop.

Limites connues

  • Certains contenus peuvent nécessiter des ajustements selon les tests (contrastes exacts selon écran).
  • Les pages légales sont simplifiées (cadre projet).

Signaler un problème

Si vous rencontrez une difficulté, contactez-nous : contact@boisnoble.fr

Nous nous engageons à proposer une solution ou une alternative.
